# prototype

A self-contained, clickable end-to-end prototype of the app — `index.html`, no build, no
dependencies (opens in any browser; also published as a hosted Artifact).

**Flows covered** (drawn from `docs/04-ux-flows.md`): 60-second onboarding → the first rep →
a day on track (check-in + the consistency chain) → **the morning after a miss** → **the
comeback ("that's a save")** → coaching-not-reminders chat → the 5-card weekly review →
accountability styles (Coach / Firm / Minimal / Savage-locked).

**Design intent:** a calm nocturnal companion — warm ember on deep still water — the
anti-Overlord. The interface is system-sans; the coach's voice is set in a serif so it reads
as a person, not UI. It's a design prototype (static, scripted), not the real app.

Open `index.html` directly, or use the scene list on the left to jump between moments.
